  • Tuesday Jan. 7, 6-9pm: Code for DC meetup. IMPORTANT: if you are looking for training, I maintain that attending meetups like Code for DC is one of the best ways to learn. This is the DC chapter of the Code for America Brigade. We are a group of volunteer civic hackers working together to solve local issues and help people engage with the city. We welcome all tech skill levels. RSVP LINK
  • Saturday Jan. 11: DC Internet Freedom Hackathon Come help make it possible for Syrian citizen reporters, Mexican journalists, and Iranian political activists get their voices heard safely. CommunityRED, with OpenITP, is holding a hackathon to make security tools easier to use, so they need tech skills ranging from folks who have a keen eye and can evaluate how easy the site is to use to UX experts to developers who code in a variety of languages. RSVP LINK
  • Monday Jan. 13, 6:30-9pm: DC Hack and Tell This is a great new meetup. Anyone can present a project; you get 5 minutes to present and then 5 minutes for Q&A. I haven't presented yet, but I love to hear what folks are working on. RSVP LINK
  • Weekend of Friday Jan 17: PDF Liberation Hackathon Extracting information from PDFs is a problem. Sunlight Foundation is organizing this hackathon to improve open source tools for PDF extraction. RSVP LINK
  • Wednesday Jan. 22, 6-9pm: Code for DC So nice, I listed it twice. This is our second meetup of the month. No RSVP link yet.
